
# 🎵 TuneFlow

TuneFlow is a modern iOS music discovery app that lets users explore artists and albums with a smooth, intuitive interface.

## ✨ Features

- 🎨 Browse top artists and albums
- 🔍 Real-time search with instant results
- 🖼️ Smart image caching for smooth scrolling
- 🔄 Pull-to-refresh functionality
- 📱 Clean and responsive UIKit design
- ⚡ Fast and efficient networking

## 🛠 Tech Stack

- **Language:** Swift 5.5+
- **Minimum iOS:** 15.0+
- **Concurrency:** Async/Await
- **Reactive:** Combine Framework
- **Networking:** URLSession
- **Caching:** NSCache
- **Architecture:** MVVM
